Este robô consiste em automatizar o processo de captura de dados na web e armazená-los em uma planilha de excell:

os dados captados serão:

nome de empresas em blumenau 

data de abertura 

faturamento

estes dados deverão ser armazenados cada um em sua respectiva coluna
exemplo:

nome        data_abertura       porte

empresa_X   11/02/1995          mei
empresa_y   15/03/1990          mei
